# SmartWatt
## Descrição do Projeto:
O SmartWatt é uma solução de análise e otimização de consumo energético voltada para ambientes corporativos e produtivos.
Através da análise de dados reais de consumo de energia e condições climáticas, o sistema identifica padrões de desperdício, gera alertas automáticos e propõe soluções sustentáveis que reduzem custos e o impacto ambiental.

O projeto conecta tecnologia, sustentabilidade e o futuro do trabalho — promovendo eficiência, automação e responsabilidade energética.

### Objetivos:
- Analisar o consumo energético e o impacto da temperatura no uso de energia.
- Identificar desperdícios e propor ações de economia.
- Demonstrar o potencial de automação e sustentabilidade em ambientes produtivos.

## Como executar o projeto:
1. Clonar o repositório:
```
git clone https://github.com/mig-2505/SmartWatt.git
cd SmartWatt/source
```
2. Instalar as dependências:
```
pip install -r requirements.txt
```
3. Adicionar os dados:

Coloque os arquivos de dados dentro da pasta /data/:
```
- building_consumption.csv
- weather_data.csv
```

## Dados do Projeto SmartWatt

A pasta data/ deve conter os arquivos CSV de entrada:

- building_consumption.csv  (dados de consumo energético)
- weather_data.csv          (dados meteorológicos)

[Link para download](https://www.kaggle.com/datasets/pythonafroz/solar-power-generation-and-energy-consumption-data?resource=download-directory)

Esses arquivos não são incluídos no repositório por ultrapassarem o limite do GitHub.
Para executar o projeto, adicione-os manualmente nesta pasta.

## Conexão com o Futuro do Trabalho
O SmartWatt demonstra como tecnologias acessíveis podem transformar ambientes de trabalho, tornando-os:

- Mais eficientes, reduzindo desperdício de energia;
- Mais sustentáveis, através da análise de dados em tempo real;
